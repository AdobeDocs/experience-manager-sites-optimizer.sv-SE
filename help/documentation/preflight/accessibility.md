# Möjlighet till preflight-tillgänglighet

![Preflight-tillgänglighetsmöjlighet](./assets/accessibility/hero.png){align="center"}

Med möjligheterna till preflight-tillgänglighet i Sites Optimizer kan du identifiera och åtgärda tillgänglighetsproblem på din webbplats för att säkerställa att den kan användas av alla, även personer med funktionshinder. Denna möjlighet fokuserar på att utvärdera webbplatsens kompatibilitet med tillgänglighetsstandarder och ger användbara rekommendationer om förbättringar.
