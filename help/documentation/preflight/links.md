# Möjlighet till preflight-länkar

![Möjlighet till preflight-länkar](./assets/links/hero.png){align="center"}

Med möjligheterna med preflight-länkar i Sites Optimizer kan du identifiera och åtgärda problem som rör användningen av länkar på din webbplats. Den här möjligheten fokuserar på att utvärdera länkstrukturen på webbplatsen och ger användbara rekommendationer om förbättringar.
