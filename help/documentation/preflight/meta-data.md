# Möjlighet till preflight-metadata

![Preflight-metadatamöjlighet](./assets/metadata/hero.png){align="center"}

Med möjligheterna att använda preflight-metadata i Sites Optimizer kan du identifiera och åtgärda problem som rör användningen av metadata på din webbplats. Den här möjligheten fokuserar på att utvärdera webbplatsens metadatastruktur och ger användbara rekommendationer om förbättringar.
