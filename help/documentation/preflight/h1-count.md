# Möjlighet att räkna preflight H1

![Affärsmöjlighet för antal H1-preflight](./assets/h1-count/hero.png){align="center"}

Affärsmöjligheterna med att räkna ut H1 i Sites Optimizer hjälper dig att identifiera och åtgärda problem med användningen av H1-rubriker på din webbplats. Den här möjligheten fokuserar på att utvärdera webbplatsens struktur och ger användbara rekommendationer om förbättringar.
