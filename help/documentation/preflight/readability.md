# Möjlighet till preflight-läsbarhet

![Möjlighet att läsa preflight](./assets/readability/hero.png){align="center"}

Med möjligheten att läsa preflight i Sites Optimizer kan du identifiera och åtgärda problem som rör läsbarheten för webbplatsens innehåll. Den här möjligheten fokuserar på att utvärdera webbplatsens innehållsstruktur och ger användbara rekommendationer om förbättringar.

Möjligheter till preflight-läsning finns på flera språk, bland annat:

* Engelska
* Franska
* Spanska
* Italienska
* Tyska
* Nederländska

Det garanterar att teamen kan utvärdera och optimera läsbarheten för innehåll inom ett brett spektrum av målgrupper.
